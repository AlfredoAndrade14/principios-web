5. Considere o cenário em que você está trabalhando em um projeto
de site com sua dupla. Qual seria sua abordagem inicial para
organizar as tarefas, escolher ferramentas e implementar boas
práticas de versionamento e desenvolvimento?

R: Para organizar o projeto, começaria definindo o escopo, as tarefas e os prazos, colocando os mesmos no GitHub Projects, para possuir uma melhor organização e acompanhamento dos dois membros do projeto, em seguida dividindo as atividades com base nas habilidades de cada um. Utilizarei Node.js para o backend e React no frontend, garantindo que ambos configurem o mesmo ambiente. Em adição, por serem frameworks da linguagem JavaScript, que é uma linguagem de grande suporte e muito conhecida na comunidade e por ambos possuírem conhecimento, além da linguagem, é a principal no desenvolvimento web que é o contexto do projeto. Centralizo o código no GitHub, seguindo boas práticas de versionamento com branches (main, develop, feature). Nos comunicando via WhatsApp para alinhamentos rápidos e resoluções de dúvidas, realizando reuniões semanais para revisões de progresso. Adotaremos padrões de código com ESLint e Prettier, documentando as instruções no README.md e implementando code reviews para manter a qualidade do projeto.
